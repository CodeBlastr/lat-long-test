# Installation

*Prerequisite*
You must have [Docker Desktop](https://www.docker.com/products/docker-desktop/) installed. 

### Steps
1. Clone the project
2. Run `docker-compose up --build -d` in Terminal
3. visit localhost:8080 and the app should be running
